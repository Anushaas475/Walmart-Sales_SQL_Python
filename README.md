# Walmart-Sales_SQL_Python

This project analyzes Walmart sales data to extract useful business insights.

## Project Overview

![Project Pipeline](https://github.com/Anushaas475/Walmart-Sales_SQL_Python/blob/main/Walmart%20Project.png)

## Objectives
- Analyze payment methods and sales patterns
- Identify top-performing categories and branches
- Optimize staffing, inventory, and revenue strategies

## Key Analyses
- Transactions and quantity by payment method
- Highest-rated category per branch
- Busiest day for each branch
- Category ratings by city
- Total profit by category
- Most used payment method per branch
- Sales by daily shifts
- Year-over-year revenue decline by branch

## Outcome
Helps improve decision-making using data-driven insights on sales, customer behavior, and profitability.
